# UI UX Pro Max

Use this skill for broad UI/UX decisions and full-page implementation reviews.

## Coverage

- Accessibility and semantic structure.
- Responsive layout and navigation.
- Color systems and contrast.
- Typography and readable rhythm.
- Form usability and validation.
- Loading, error, empty, and success states.
- Interaction states and motion.
- Visual consistency across pages.

## Process

- Inspect the product goal and current code first.
- Identify the highest-impact UX problems.
- Fix the simplest correct set of issues.
- Verify build quality and regression risk.
