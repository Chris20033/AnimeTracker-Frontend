# Frontend Design

Use this skill when a page needs a stronger visual direction, a new layout concept, or a product-specific aesthetic.

## Principles

- Preserve the product context before changing visuals.
- Avoid generic SaaS hero sections, repeated icon-card grids, and decorative gradient text.
- Pick a coherent palette and use semantic tokens instead of scattering raw colors.
- Make typography, spacing, and layout feel intentional at mobile, tablet, and desktop widths.
- For AnimeTracker, prefer a modern anime-inspired direction: sakura, warm amber, violet night, soft paper tones, and restrained accents.

## Workflow

- Inspect existing components, tokens, and route structure first.
- Define the page hierarchy before touching colors.
- Keep light and dark mode equally designed.
- Use CSS variables from `src/index.css` whenever possible.
- Verify responsive behavior and accessible contrast.
