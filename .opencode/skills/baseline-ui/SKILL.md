# Baseline UI

Use this skill for fast cleanup passes when UI already exists but feels rough.

## Checklist

- Align spacing rhythm across sections and components.
- Improve hierarchy before adding decoration.
- Make mobile layouts usable and avoid cramped controls.
- Check labels, focus states, hover states, disabled states, and error states.
- Use existing CSS variables and component patterns.
- Remove unnecessary wrappers, repeated classes, and visual noise.

## Output Standard

- Minimal, targeted changes.
- No broad redesign unless explicitly allowed.
- Lint and build should pass after changes.
