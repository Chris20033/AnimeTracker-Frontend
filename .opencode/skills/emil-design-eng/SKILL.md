# Emil Design Engineering

Use this skill when interface craft matters: interaction feel, animation, hierarchy, and component details.

## Principles

- Make interfaces feel calm, precise, and responsive.
- Prefer a few excellent details over many decorative effects.
- Animation should explain state change, not perform for attention.
- Interactions should feel immediate and reversible.
- Components should be simple, composable, and easy to maintain.

## Implementation Notes

- Use CSS transitions for small hover/focus/pressed states.
- Avoid heavy animation libraries unless already present.
- Keep layout stable during loading and mutation states.
- Design error and success feedback with the same care as happy paths.
